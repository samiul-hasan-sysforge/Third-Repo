See how to push code from Vs Code to GitHub Repository For First Time

  1. git init
  2. git add .
    • git add (file/folder name)
  3. git commit -m "your commit"
  4. git branch - M main
    • In future we will change the main branch
  5. git remote add origin your repository url
  6. git push -u origin main
    • main branch will be changed if we change it in 4th step

This is th method for the second time or update the code

  1. git add .
  2. git commit -m "second commit"
  3. git push -u origin main
    • git push (it also work)
